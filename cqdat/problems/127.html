Problem 127: Large Multiples

This practice problem was made by CodeRams Co-President Plourde27
This problem has a difficulty of 70


For numbers 1 <= n <= 3, the lowest number that is divisible by all numbers less than or equal to n is simply the factorial of n. However, for n > 3, that is not always the case. For example, the number 2520 (2 * 2 * 2 * 3 * 3 * 5 * 7) is the lowest number that is divisible by all numbers from 1 to 10.

In this problem, you are asked to find the lowest number that divides all of the numbers from 1 to a given number “n”.

Program Input


The first line of Prob103.in.txt will contain a positive integer T indicating the number of test cases. The following T lines each contain a positive integer less than 1000.

Example Input:


7
1
2
3
10
15
30
100

Program Output


For each number “n”, output the lowest number that evenly divides all numbers 1 to “n”. Because the number can be very large, print the remainder when dividing the number by 10^9 + 7

Example Output:


1
2
6
2520
360360
89546497
485679014
