Problem 128: Multiple of 7

This practice problem was made by CodeRams Co-President Xavier Plourde
This problem has a difficulty of 0



For numbers such as 2, 3, and 5, there exists an easy rule to figure out if they evenly divide another number or not. However, for the number 7 no simple divisibility rule exists.
In this problem, write a computer program to determine whether or not a number is a multiple of 7

Program Input


The only line of the input will contain a positive integer N not greater than 1000.

Example Input:


14

Program Output


For the number "N", print YES if it is a multiple of 7, otherwise print NO.

Example Output:


YES
