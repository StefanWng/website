Problem 129: Grades

This practice problem was made by CodeRams Co-President Xavier Plourde
This problem has a difficulty of 5



In most schools, students are given letter grades based on their numerical performance in a class.
In this problem, write a computer program to find the letter grade of a student, given their numberical grade out of 100.

Program Input


The only line of the input will contain a positive integer N between 1 and 100, representing the student's numerical grade.

Example Input:


74

Program Output


For the number "N", print the corresponding letter grade:
"A" if N >= 90
"B" if 80 <= N <= 89
"C" if 70 <= N <= 79
"D" if 65 <= N <= 69
"F" if N < 65

Example Output:


C
